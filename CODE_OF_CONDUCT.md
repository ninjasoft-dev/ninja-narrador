# Código de conduta

Esperamos colaboração profissional, respeitosa e orientada ao mérito técnico.
Não são aceitos assédio, discriminação, exposição de dados pessoais, conteúdo
abusivo ou uso do projeto para personificação sem consentimento.

Mantenedores podem editar ou remover contribuições incompatíveis com estas
regras e restringir a participação de quem insistir nesse comportamento.

Relatos devem ser feitos de forma privada pelo canal de segurança do repositório.
